# Code Smells
---
10 code smells that we are supposed to look out for in the project
- Magic numbers
- Global variables
- Bad names
- Unhelpful comments
- Too many arguments in a function
- A function that is way too long
- Redundant code
- Overly complex decision tree
- Spaghetti code
- Dead code
